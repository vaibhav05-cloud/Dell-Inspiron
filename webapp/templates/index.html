🧭 Document GraphRAG

Upload a PDF, ingest it into a vector + knowledge graph index, and ask scoped questions.

Ingestion pipeline

PDF Manual → Parsing → sample_parsed.json
→ Vision/Table Enrichment → sample_processed.json
→ Semantic Chunking → chunks.json
→ Entity Extraction → entities.json
→ Relationship Extraction → relationships.json
→ FAISS Index + Neo4j Graph

Query pipeline

User Query → Query Planner Agent
→ Semantic Retrieval (FAISS) + Graph Retrieval (Neo4j)
→ Fusion Agent → Re-ranking Agent → Evidence Agent
→ Context Builder Agent → Answer Synthesis
→ JSON Output (in-scope or out-of-scope)

Use the sidebar to upload a document, ask questions, explore the knowledge graph, or view the pipeline dashboard.
